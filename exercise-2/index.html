Welcome to Module 18 Exercise 2

To complete this exercise, edit your main.js file. You'll get practice with some basic D3 data binding.

  • Create a varaible myDiv by selecting the element with id my-div (this is already created for you).
  • Create a variable paragraphs by selecting all p elements in myDiv, and binding the data people to it. Make sure to specify the name attribute as an identifying characteristic in your data-join.
  • You'll then want to add a paragraph to the div for each person in your dataset. To do this:
    • Determine which elements are in the dataset but are not represented in the DOM (enter)
    • For each new element, append a paragraph (p) element
    • For each paragraph, set the text of the paragraph to the name of the person (text)
    • For each paragraph, set the color to the favColor of the person (color)
  • Then, inside the setTimeout function that waits 3 seconds, do the following
    • Re-define your variable paragraphs by selecting all p elements in myDiv, and binding the data people2 to it. Make sure to specify the name attribute as an identifying characteristic in your data-join. Note, you cannot use the selection from earlier, as new elements have been added since that selection was defined
    • Determine which people are no longer represented in the data (exit), and transition their opacity to 0 and remove them.
